A production-ready **Payload CMS + Next.js starter template**.

Built for modern full-stack web applications using:

* Payload CMS
* Next.js
* PostgreSQL
* Tailwind CSS
* TypeScript

This starter provides a strong foundation for building:

• headless CMS platforms
• SaaS applications
• content-driven websites
• API-first platforms

# Getting Started

Clone the repository:

```
git clone https://github.com/sazmco/payload-nextjs-starter
```

Install dependencies:

```
pnpm install
```

Run development:

```
pnpm dev
```
